RxJava 实现事件通知机制

09AE9C's Blog
2018-04-08
Kotlin

使用 RxJava 实现应用内的事件分发，无需借助第三方库，少量的代码便能实现所需的功能

结构

整体结构很简单，分为

  • EventType: 定义事件类型类型
  • Event: 具体事件模型
  • EventBus: 事件总线

最初的版本对事件定义采用了 接口 + 枚举类的方式实现，每一个事件类型就是一个枚举类型，后来发现这种方式不太方便对一类事件的注册与监听，必须手动添加所有事件，于是改成了用 Int 类型定义事件类型，这样如果需要注册某一类事件，只需整理对应的事件到对应 Int 类型集合里即可实现，下面是具体实现过程
