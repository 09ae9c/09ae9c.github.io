Dart 基础

09AE9C's Blog
2019-01-06
Dart

重要概念

  • Dart 中一切皆为对象(numbers，functions，null)，每一个对象都是一个类的实例，所有对象都继承自 Object 类
  • Dart 是强类型语言，但类型声明是可选的，因为 Dart 有类型推断，能根据上下文推断出具体类型，如果不明确具体类型，可以用 dynamic 关键字修饰(类似 Kotlin 中的 Any)
  • Dart 支持泛型
  • Dart 支持顶层方法，以及静态方法和类方法，方法内也可以创建方法(嵌套方法)
  • Dart 支持顶层变量，以及静态变量和类变量
  • Dart 没有访问权限控制关键字(public，protected，private)，如果方法名或变量名前缀是_ 则表示是 library 级别私有
  • Dart 有表达式和语句，表达式具有运行时类型，而语句没有类型

变量

变量声明符有 Object，dynamic，var 以及具体类型，其中 Object 和 dynamic 都表示所有类型，关于他们的区别，参考这段话：

Another perspective on dynamic is that it’s not really a type - it’s a way to turn off type checking and tell the static type system “trust me, I know what I’m doing”. Writing dynamic o; declares a variable that isn’t typed - it’s instead marked as “not type-checked”.
When you write Object o = something; you are telling the system that it can’t assume anything about o except that it’s an Object. You can call toString and hashCode because those methods are defined on Object, but if you try to do o.foo() you will get a warning - it can’t see that you can do that, and so it warns you that your code is probably wrong.
If you write dynamic o = something you are telling the system to not assume anything, and to not check anything. If you write o.foo() then it will not warn you. You’ve told it that “anything related to o is OK! Trust me, I know what I’m doing”, and so it thinks o.foo() is OK.
With great power comes great responsibility - if you disable type-checking for a variable, it falls back on you to make sure you don’t do anything wrong.

而 var 关键字申明的变量，就是交给 Dart 的类型推断系统自动推断具体类型，申明的时候不确定类型，在赋值的时候推断出具体的类型

默认值

变量都有默认值，一个变量在申明后不赋值，则它的默认值是 null，就算是数字类型(numeric)，默认值也是 null，因为在 Dart 中一切皆是对象

Final 和 Const 关键字

加了 final 关键字修饰的变量，一旦赋值就不可再被修改，而加了 const 关键字修饰的变量是一个编译器常量(const 修饰的变量默认是 final)

基本数据类型

  • numbers(int, double)
  • strings(String)
  • booleans(bool)
  • lists(List) 注: 在 Dart 中，数组(arrays)也是 List 对象
  • maps(Map)
  • runes(the UTF-32 code points of a string)
  • symbols(A Symbol object represents an operator or identifier declared)

类

Dart 是一门混合继承的面向对象语言，每一个对象都是一个类的实例，所有类都继承自 Object，混合继承指虽然每个类只有一个父类，但类体可以在多个类层次结构中重用

访问类成员的方式与 Kotlin 类似，有 . 号 和 ?. 号，?. 会做空检查，如果被访问对象是空，则不会触发后面的调用

类的构造

在 Dart2 中可以省略 new 关键字

常量构造器

用于创建一个编译期的常量，用 const 关键字修饰

命名构造器(Named Constructor)

类似定义类方法创造类实例，可以创建多种不同需求的构造器

命名构造器不能被继承，在父类中的命名构造器在子类中不能使用
